# Mumbai House Price Prediction Model

- It is a Machine learning Model use to predict the price of houses located in the city of Mumbai. Based on the several parameters like longitude, latitude, no.of rooms, total no.of Bedrooms, etc , it predict the most accurate Price of the house.

- It uses Supervised Learning (Random Forest Regressor) model to predict the house price in a most precise way.

- This ML project includes the concepts like Python, NumPy, Pandas, Scikit-learn, Flask, HTML, CSS, JavaScript.
